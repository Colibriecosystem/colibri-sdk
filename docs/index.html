enter port + token

Open this file directly, paste the Port + Access token from the terminal (Settings → Program → Local API), and hit Connect — the values are remembered in this browser. The terminal must have “Allow web browser access” ticked (that lets any browser page reach the API; the token still guards it). Then expand an endpoint and hit Send.
